Terminal Simulado

Pratique comandos essenciais de Kubernetes sem precisar de um cluster. Este terminal simula respostas realistas para comandos comuns.

kubectl-terminal — bash
Bem-vindo ao Terminal Simulado de Kubernetes!
Digite um comando kubectl, helm ou digite "help" para ver os comandos disponiveis.
kubectl get pods kubectl get nodes kubectl get svc kubectl get deployments kubectl describe pod web-0 kubectl logs web-0 kubectl apply -f app.yaml kubectl rollout status deploy/web kubectl rollout undo deploy/web helm list
user@k8s-lab:~$
Dica: Tente comandos como kubectl get pods -A, kubectl describe pod web-0, ou kubectl logs frontend-abc123. Pressione Enter para executar e seta para cima/baixo para historico.

[00:00] Simulador pronto. Clique em "Deploy v2" para iniciar um rolling update.
Como funciona: O Kubernetes substitui Pods antigos por novos gradualmente (controlled by maxSurge e maxUnavailable). Observe como os Pods sao criados e encerrados um a um, garantindo zero downtime.
